# SAP Vision One – Mobile App for SAP SuccessFactors

**A cross-platform mobile app** built with Xamarin for Android and iOS, integrating with SAP SuccessFactors for HR management and Azure DevOps for workflow automation.

This was a personal project to explore mobile integration with enterprise systems — enabling users to access HR data, manage tasks, and track projects on the go.

## Features

- Real-time access to SAP SuccessFactors HR modules (e.g., employee profiles, time tracking, performance reviews)
- Azure DevOps integration for task management, bug tracking, and CI/CD status updates
- Cross-platform UI with Xamarin.Forms for consistent Android/iOS experience
- Secure authentication (OAuth/SAML) with SAP and Azure APIs
- Push notifications for HR approvals and DevOps alerts
- Offline mode with data sync when online

<br>

## Built With

- **Xamarin** (C#/.NET for Android & iOS)
- SAP SuccessFactors APIs (REST/OData for HR data)
- Azure DevOps APIs (for boards, repos, pipelines)
- Xamarin.Essentials (cross-platform features like notifications)
- SQLite.NET for local storage/offline support
